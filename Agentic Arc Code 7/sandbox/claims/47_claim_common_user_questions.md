# Common User Questions on Claims

Example user questions the AI agent should be able to answer with these files.

- **Why was my claim denied?**: Use denial codes, denial guidance, and missing information topics.
- **What does this claim status mean?**: Use status dictionary and event files.
- **Was my claim processed automatically?**: Use auto adjudication indicator and event history.
- **How much was billed, allowed, and paid?**: Use financial terms and claim metadata.
- **Who submitted the claim and how?**: Use provider, member, and submission channel context.
- **What information is still missing?**: Use denial gap files and rework fields.
- **What happened next on my claim?**: Use event sequence and lifecycle overview.
- **How can I correct or appeal the claim?**: Use rework and appeal tracking files.
