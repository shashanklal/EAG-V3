# Claim Create Features - Core Identifiers and Header Data

Simple descriptions of key data elements used when a claim is created by a member or provider.

## member_id
Unique ID of the person covered by the health plan.

## provider_id
Unique ID of the doctor, hospital, lab, or facility sending the claim.

## group_id
Employer or group contract tied to the member plan.

## policy_id
Policy or benefit package ID that defines coverage rules.

## claim_id
Unique claim number assigned to the full claim.

## claim_type
Type of claim such as Professional, Institutional, Dental, or Pharmacy.

## svc_from_date
Start date when service was given.

## svc_to_date
End date when service was given.

## line_id
Unique ID for one claim line.

## line_num
Sequence number of the line inside the claim.

## proc_code
Procedure or service code billed by the provider.

## rev_code
Revenue code used mainly on facility or institutional billing.
