# Claim Sample Query Patterns

Intent examples useful for an AI agent working on claim questions.

- **status inquiry**: What is the current status of claim CLM202600010?
- **payment inquiry**: How much was paid on claim CLM202600003?
- **denial inquiry**: Why was line 2 on claim CLM202600002 denied?
- **timeline inquiry**: Show the key events for claim CLM202600051.
- **rework inquiry**: Did claim CLM202600001 go into rework and why?
- **lookup inquiry**: What does denial code CO-16 mean?
- **coding inquiry**: What is procedure code 99214?
- **metadata inquiry**: Which file contains event sequence information?
